# Titanic-Eda-Analysis
Titanic EDA using Python to uncover survival patterns across passenger class, gender, and age.
# Titanic EDA Analysis

## 📌 Objective

To explore factors affecting passenger survival using the Titanic dataset.

## 🔍 Key Insights

* Higher-class passengers had better survival rates.
* Females had significantly higher survival chances.
* Age distribution varied across passenger classes.
* Socio-economic status influenced survival outcomes.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib / Seaborn

## 📊 Techniques

* Crosstab analysis
* Data visualization
* Basic feature understanding
